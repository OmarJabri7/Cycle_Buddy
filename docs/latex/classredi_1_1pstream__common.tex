Class template for common base class. 

Abstract Base Class providing common functionality for \mbox{\hyperlink{classredi_1_1basic__ipstream}{basic\+\_\+ipstream}}, \mbox{\hyperlink{classredi_1_1basic__opstream}{basic\+\_\+opstream}} and \mbox{\hyperlink{classredi_1_1basic__pstream}{basic\+\_\+pstream}}. \mbox{\hyperlink{classredi_1_1pstream__common}{pstream\+\_\+common}} manages the \mbox{\hyperlink{classredi_1_1basic__pstreambuf}{basic\+\_\+pstreambuf}} stream buffer that is used by the derived classes to initialise an iostream class.

Pure virtual destructor. 

This is a pure virtual function to make {\ttfamily \mbox{\hyperlink{classredi_1_1pstream__common}{pstream\+\_\+common}}} abstract. Because it is the destructor it will be called by derived classes and so must be defined. It is also protected, to discourage use of the P\+Streams classes through pointers or references to the base class.
